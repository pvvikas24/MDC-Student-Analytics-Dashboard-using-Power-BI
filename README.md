📊 MDC Student Analytics Dashboard (Power BI)

An interactive Student Analytics Dashboard built using Power BI to analyze student distribution across batches, departments, domains, and roles. This project enables data-driven decision-making through intuitive and dynamic visualizations.

👤 Author

PV Vikas
DataVerse AY 2025–2026

🚀 Project Overview

This dashboard provides a comprehensive view of student data within MDC, helping stakeholders understand trends, demographics, and distributions efficiently.

It focuses on:

Monitoring total student count
Analyzing batch-wise growth trends
Visualizing gender distribution
Exploring department, domain, and role segmentation
📌 Key Features
📈 Total Students KPI
Displays the total number of students (147)
📊 Batch-wise Trend Analysis
Shows growth and distribution across academic years
👥 Gender Distribution
Visual breakdown using donut charts
🧩 Interactive Filters (Slicers)
Allows users to dynamically filter data by:
Batch
Department
Domain
Role
📋 Detailed Table Matrix
Provides granular insights into student records
